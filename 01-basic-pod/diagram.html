🏗️ Components

Pod: Basic deployment unit containing a single nginx container
Container: nginx web server running on port 80
Scope: Demonstrates fundamental Kubernetes pod concept

🎯 Key Concepts

Pods: Smallest deployable units in Kubernetes
IP Address: Each pod gets its own IP address
Network: Containers in a pod share the same network namespace

📋 Deployment Commands

# Deploy the pod
kubectl apply -f 01.pod.yml

# Check pod status
kubectl get pods

# Describe the pod
kubectl describe pod myapp-pod

# Delete the pod
kubectl delete -f 01.pod.yml
